# QA Checklist for Main Owner Launch

## 1. Documentation Review
- [ ] One-line value proposition is clear and conversion-focused.
- [ ] 3-step setup guide is accurate and easy to follow.
- [ ] 5 FAQ answers cover common user concerns.
- [ ] "What to expect" checklist provides a clear value summary.

## 2. Visual & Design Verification
- [ ] Hero card, Compact card, and Mobile tile are visually consistent.
- [ ] All copy passes the 4.5:1 contrast ratio for accessibility.
- [ ] Cards display correctly on both desktop and mobile viewports.
- [ ] Hover and active states are implemented and visually distinct.

## 3. Implementation Check
- [ ] Documentation page is accessible via the main navigation.
- [ ] Promotional card is prominently featured on the Dashboard.
- [ ] Analytics events (start, complete, view, click) are correctly named and payloaded.
- [ ] Semantic HTML recommendations are followed for accessibility.

## 4. Final Sign-off
- [ ] All tests in `tests/` pass without regressions.
- [ ] Product Owner review and approval.
- [ ] Design review and approval.
- [ ] Support and Marketing notified.
